# Letter-Drawer
A Java console application that dynamically draws complex letters (W and Z) using asterisks based on user-defined sizes. It features robust input validation (preventing scanner bugs and enforcing size constraints) and demonstrates advanced nested loop algorithms for console-based graphics.
